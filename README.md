# NQueens
Solution of N-Queens problem by Recursive Backtracking. Program produces solution by printing it console such as : 

```
Q X X X X X X X
X X X X X X Q X
X X X X Q X X X
X X X X X X X Q
X Q X X X X X X
X X X Q X X X X
X X X X X Q X X
X X Q X X X X X
```
